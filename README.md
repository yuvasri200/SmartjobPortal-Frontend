# Smart Job Portal - Frontend

## 🚀 Project Overview
Smart Job Portal Frontend is built using HTML, CSS, and JavaScript.
It connects to the Spring Boot backend using REST APIs.

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## 📌 Features
- User Registration Page
- Login Page
- Job Listing Page
- Apply Job Button
- Logout Functionality
- REST API Integration

## 📂 Project Files
index.html → Home Page  
login.html → Login Page  
register.html → Register Page  
jobs.html → Job Listing Page  
style.css → UI Styling  
script.js → API Integration  

## 🔗 Backend API URL
http://localhost:8080/api

## ▶️ How to Run
1. Make sure backend is running
2. Open index.html in browser
3. Register → Login → View Jobs → Apply
